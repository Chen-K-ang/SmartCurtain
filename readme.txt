四个基本按键
	设置模式：RC手动，Auto自动，Time定时
	设置定时：每次按下可以在闪烁部位通过加减键改变定时时间
	加：非自动模式下改变窗帘位置（向全开方向，电机正转）
	减：非自动模式下改变窗帘位置

两个模拟按键
	模拟声音：分关、半开、全开
	模拟限位开关：任意模式下按下，在电机旋转时可将其停止
                                        随后模式转化为RC手动

LCD1602显示
	默认显示：第一行：当前时间
		第二行：s:期望位置、g:检测位置、模式
	定时显示：在定时模式下，第二行，每2s会交替显示定时时间和默认显示的第二行

光敏
	在亮时，窗帘为全关
